# Aplicativo Interactivo de Lógica y Algoritmos

## Descripción
Programa desarrollado en C++ que permite:
- Operaciones matemáticas básicas
- Registro de 5 notas
- Cálculo de promedio, mayor y menor
- Conteo de aprobados y reprobados
- Guardado de resultados en archivo txt

## Lenguaje utilizado
C++

## Autor
Gina Quevedo

src -> código fuente
resultados -> archivo generado
capturas -> evidencias
